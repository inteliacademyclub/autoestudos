# Function Calling / Tool Use

O *Function Calling* (ou Uso de Ferramentas) é um recurso avançado que permite que a Inteligência Artificial saia da sua "caixa" de texto e interaja com o mundo real e com sistemas externos.

Através dele, a IA pode decidir executar funções programadas (como fazer uma pesquisa na web, consultar um banco de dados privado, enviar um e-mail ou rodar um código Python) durante a conversa para obter informações atualizadas e dar uma resposta mais rica e exata ao usuário.
