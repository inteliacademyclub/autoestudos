# Temperatura

A Temperatura é uma configuração que controla o nível de aleatoriedade e criatividade nas respostas geradas por uma IA.

- **Temperatura baixa (ex: 0.1 - 0.3):** Respostas mais diretas, focadas, lógicas e previsíveis. Ideal para programação, análise de dados e respostas factuais.
- **Temperatura alta (ex: 0.7 - 1.0):** Respostas mais criativas, variadas e imprevisíveis. Ideal para escrita de histórias, *brainstorming* e poemas.
