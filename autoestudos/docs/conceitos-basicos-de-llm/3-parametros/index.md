# Parâmetros

Parâmetros são as "conexões neurais" de um modelo de Inteligência Artificial. Eles definem o conhecimento, a memória e a lógica da IA, sendo ajustados durante a fase de treinamento.

Quando ouvimos que um modelo tem "70 bilhões de parâmetros" (70B), isso significa que ele possui 70 bilhões de variáveis matemáticas usadas para prever qual será a próxima palavra gerada. Em geral, mais parâmetros significam maior capacidade de raciocínio, mas também exigem mais poder computacional.
