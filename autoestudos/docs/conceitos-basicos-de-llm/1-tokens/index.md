# Tokens

Tokens são as unidades fundamentais de dados processadas por um modelo de linguagem (LLM). Em vez de ler palavras ou letras individualmente, a IA divide o texto em "pedaços" chamados tokens.

Um token pode ser uma palavra inteira (como "carro"), uma sílaba ou apenas uma letra. Em média, no idioma inglês, 1 token equivale a cerca de 4 caracteres. Em português, devido à acentuação, o número de tokens gerados pode ser ligeiramente maior.
