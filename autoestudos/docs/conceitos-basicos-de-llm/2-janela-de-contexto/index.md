# Janela de Contexto

A janela de contexto (Context Window) é a quantidade máxima de tokens que um modelo de IA consegue "lembrar" ou processar de uma só vez durante uma interação. 

Isso inclui tanto a sua pergunta (prompt) quanto a resposta da IA. Se uma conversa ultrapassar o limite da janela de contexto (por exemplo, 128 mil tokens), a IA começará a "esquecer" as mensagens mais antigas para dar espaço às novas, perdendo o contexto inicial da conversa.
